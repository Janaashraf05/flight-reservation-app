# ✈️ Flight Reservation System  

A **desktop app** built with:  
- Python + Tkinter (GUI)  
- SQLite (Database)  
- PyInstaller (EXE)  

## 🚀 Features  
- Book flights  
- View all reservations  
- Edit/Delete bookings  

## 🛠️ How to Run  
1. **For EXE**:  
   - Download the `dist` folder  
   - Double-click `main.exe`  

2. **For Developers**:  
   ```bash
   pip install pyinstaller
   python main.py
